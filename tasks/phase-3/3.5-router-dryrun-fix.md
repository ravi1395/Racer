# Task 3.5 — Router Dry-Run Fix

**Phase:** v1.6.0  
**Risk:** Low  
**Effort:** ~1 day  
**Breaking:** No (old method deprecated, not removed until v2.0.0)

## Problem

`RacerRouterService.dryRun(Object payload)` only evaluates rules with `source = PAYLOAD`. Rules with `source = SENDER` or `source = ID` are silently skipped, giving developers a false impression of how a real message would be routed.

## File to Change

### `router/RacerRouterService.java`

**Step 1:** Add an overloaded `dryRun` method that accepts all three possible rule-source values:

```java
/**
 * Evaluates all routing rules against a synthetic message.
 * Unlike {@link #dryRun(Object)}, this variant correctly evaluates SENDER and ID rules too.
 *
 * @param payload   message payload object (evaluated against PAYLOAD rules); may be null
 * @param sender    sender label string (evaluated against SENDER rules); may be null to skip SENDER rules
 * @param messageId message ID string (evaluated against ID rules); may be null to skip ID rules
 * @return the first matching {@link RouteDecision}, or {@code null} if no rule matches
 */
public RouteDecision dryRun(Object payload, @Nullable String sender, @Nullable String messageId) {
    for (CompiledRouteRule rule : globalRules) {
        boolean matched = switch (rule.getSource()) {
            case PAYLOAD  -> payload != null   && rule.test(objectMapper.valueToTree(payload));
            case SENDER   -> sender != null    && rule.test(sender);
            case ID       -> messageId != null && rule.test(messageId);
        };
        if (matched) {
            return rule.getDecision();
        }
    }
    return null;
}
```

**Step 2:** Deprecate the original single-argument method:

```java
/**
 * @deprecated Use {@link #dryRun(Object, String, String)} to correctly evaluate all rule sources.
 *             This method silently skips SENDER and ID rules.
 *             Will be removed in v2.0.0.
 */
@Deprecated(since = "1.6.0", forRemoval = true)
public RouteDecision dryRun(Object payload) {
    return dryRun(payload, null, null);
}
```
