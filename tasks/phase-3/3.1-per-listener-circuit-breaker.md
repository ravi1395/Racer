# Task 3.1 — Per-Listener Circuit Breaker Configuration

**Phase:** v1.6.0  
**Risk:** Low  
**Effort:** ~3 days  
**Breaking:** No

## Problem

All listeners share one global `racer.circuit-breaker.*` config. An order-processing listener (where 10% failure is critical) and an analytics listener (where 30% is acceptable) can't have different thresholds.

## Files to Change

### 1. `config/RacerProperties.java`

Add per-listener overrides inside the existing `CircuitBreakerProperties` inner class:

```java
@Data
public static class CircuitBreakerProperties {

    // ... existing global fields ...

    /**
     * Per-listener overrides. Each key is a listener ID (from {@code @RacerListener(id="…")}
     * or the auto-generated {@code "<beanName>.<methodName>"}).
     *
     * Example:
     * <pre>
     * racer.circuit-breaker.listeners.order-listener.failure-rate-threshold=10
     * racer.circuit-breaker.listeners.order-listener.sliding-window-size=20
     * racer.circuit-breaker.listeners.analytics-listener.failure-rate-threshold=40
     * </pre>
     *
     * Any property not overridden falls back to the global default.
     */
    private Map<String, ListenerCircuitBreakerOverride> listeners = new LinkedHashMap<>();

    @Data
    public static class ListenerCircuitBreakerOverride {
        private Integer failureRateThreshold;
        private Integer slidingWindowSize;
        private Integer waitDurationInOpenStateSeconds;
        private Integer permittedCallsInHalfOpenState;
    }
}
```

### 2. `circuitbreaker/RacerCircuitBreakerRegistry.java`

In `getOrCreate(String listenerId)`, merge per-listener overrides with global defaults:

```java
public RacerCircuitBreaker getOrCreate(String listenerId) {
    return breakers.computeIfAbsent(listenerId, id -> {
        CircuitBreakerProperties global = racerProperties.getCircuitBreaker();
        ListenerCircuitBreakerOverride override = global.getListeners().get(id);

        int threshold = override != null && override.getFailureRateThreshold() != null
                ? override.getFailureRateThreshold()
                : global.getFailureRateThreshold();
        int windowSize = override != null && override.getSlidingWindowSize() != null
                ? override.getSlidingWindowSize()
                : global.getSlidingWindowSize();
        int waitSecs = override != null && override.getWaitDurationInOpenStateSeconds() != null
                ? override.getWaitDurationInOpenStateSeconds()
                : global.getWaitDurationInOpenStateSeconds();
        int halfOpenProbes = override != null && override.getPermittedCallsInHalfOpenState() != null
                ? override.getPermittedCallsInHalfOpenState()
                : global.getPermittedCallsInHalfOpenState();

        return new RacerCircuitBreaker(id, windowSize, threshold,
                Duration.ofSeconds(waitSecs), halfOpenProbes);
    });
}
```
